<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl sm:text-3xl font-bold text-gray-900 dark:text-white mb-4">
    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with your personal Second Brain in minutes
  </p>
  
  <Steps>
    <Step title="Create your account">
      Sign up to create your personal knowledge vault. Each account gets a private, encrypted space for your content.
      
      ```bash cURL
      curl -X POST http://localhost:8000/api/v1/auth/signup \
        -H "Content-Type: application/json" \
        -d '{"username": "your-username", "password": "your-password"}'
      ```
    </Step>
    
    <Step title="Save your first content">
      Add articles, videos, tweets, or documents to your vault. MIND-f-LAYER automatically generates vector embeddings for semantic search.
      
      ```bash
      curl -X POST http://localhost:8000/api/v1/content \
        -H "Authorization: Bearer YOUR_JWT_TOKEN" \
        -H "Content-Type: application/json" \
        -d '{
          "link": "https://example.com/article",
          "type": "article",
          "title": "Introduction to RAG",
          "description": "Learn about Retrieval-Augmented Generation",
          "tags": ["ai", "rag", "learning"]
        }'
      ```
    </Step>
    
    <Step title="Query your knowledge">
      Ask questions in natural language and get AI-powered answers based on your saved content.
      
      ```bash
      curl -X POST http://localhost:8000/api/v1/brain/query \
        -H "Authorization: Bearer YOUR_JWT_TOKEN" \
        -H "Content-Type: application/json" \
        -d '{
          "query": "What have I learned about RAG?",
          "topK": 5
        }'
      ```
      
      <Accordion title="Example response">
        ```json
        {
          "message": "Query completed",
          "answer": "Based on your saved content, you've learned that RAG (Retrieval-Augmented Generation) combines semantic search with large language models...",
          "relevantContent": [
            {
              "id": 1,
              "title": "Introduction to RAG",
              "link": "https://example.com/article",
              "type": "article",
              "score": 87.3,
              "tags": ["ai", "rag", "learning"]
            }
          ]
        }
        ```
      </Accordion>
    </Step>
  </Steps>
</div>
